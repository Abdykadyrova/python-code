Атрибут form

  • Содержание
    • Синтаксис
    • Значения
    • Значение по умолчанию
    • Пример
    • Браузеры

Связывает кнопку с формой по её идентификатору. Такая связь необходима в случае, когда кнопка не располагается внутри элемента <form>, например, при создании её программно.

Синтаксис

<button form="<идентификатор>">...</button>

Значения

Идентификатор формы (значение атрибута id элемента <form>).

Значение по умолчанию

Нет.

Пример

<!DOCTYPE html>
<html>
 <head>
  <meta charset="utf-8">
  <title>button, атрибут form</title>
 </head>
 <body>
  <form id="data" action="handler.php">
   <p><input placeholder="Ваше имя" name="user"></p>
  </form>
  <p><button type="submit" form="data">Отправить форму</button></p>
 </body>
</html>
